**2020.3.5更新：关于WPS安装后菜单栏为英文的问题**

正常安装WPS完毕后，如果WPS菜单栏显示为英文，在终端安装WPS中文语言包可以解决问题：

```bash
sudo pacman -S wps-office-mui-zh-cn
```
